# modula2-cli-hello-world

## Description
A POC for modula2 programming language.

## Tech stack
- gm2

## Docker stack
- ubuntu:22.04

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
